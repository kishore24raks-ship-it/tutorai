# **App Name**: TutorAI

## Core Features:

- Secure Authentication: Enable secure user login and signup functionality.
- AI Chatbot: Implement an interactive AI chatbot that can provide explanations and answer questions.
- Custom Quiz Generator: Generate quizzes based on user-defined topics, number of questions, and difficulty levels.
- Quiz History: Maintain a record of the student's past quiz results to track their progress.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC), evoking a sense of calm and trust.
- Background color: Light gray (#F0F0F0), providing a neutral and clean backdrop.
- Accent color: Pale sky blue (#CFE2FF), for highlighting key elements without overwhelming the design.
- Body and headline font: 'PT Sans' for a friendly yet professional style.
- Use simple, outline-style icons for a clean and modern interface.
- Design a clean and intuitive layout, prioritizing ease of navigation and readability.
- Incorporate subtle animations and transitions to provide feedback and enhance user engagement.
